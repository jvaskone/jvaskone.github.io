1. Surface smoothing and reduction of the number of triangles:

The first image shows a surface generated from raw volumetric data.

It is displayed directly by OpenGL without any optimization. If the volumetric data is big, the surface contains a huge number of triangles which prevents real-time rendering and it can also contain bad-shaped (pin-like) triangles.

The second image shows a smooth surface which is done by geometry based triangulation.


The third surface is generated from the second using triangle reduction and the number of triangles are reduced approximately by 80%.


2. Property mapping:

Property mapping means that some data is interpreted as colors and is combined with or mapped to data which is represented as shape.
